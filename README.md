# 推荐链路 Demo

这是一个零依赖静态前端入门项目，主题贴近推荐算法里的精排和重排链路。

## 功能

- 模拟候选内容经过精排模型打分
- 新增零依赖 DeepFM 推理模块，包含 linear、FM interaction、DNN 三部分
- 支持调节 CTR、兴趣、新鲜度、探索比例等权重
- 支持兴趣强化、新鲜度优先、转化优先三种场景
- 展示重排后的 Feed 列表
- 显式标注类目打散、作者去重、疲劳惩罚、探索位、多样性补偿等原因
- 实时统计类目覆盖、平均最终分和探索内容数量

## 运行方式

直接用浏览器打开 `index.html`。

- 加入召回层和粗排层
